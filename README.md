# 🚀 Incomesure AI

### AI-Powered Parametric Insurance for Gig Workers

---

## 📌 Problem Statement

Gig workers such as Zomato and Swiggy delivery partners face unpredictable income loss due to external disruptions like heavy rain, extreme heat, pollution, and sudden restrictions.

These disruptions can reduce their earnings by up to **30%**, and currently, there is **no system to protect their income** during such events.

---

## 💡 Our Solution

**Incomesure AI** is an AI-powered parametric insurance platform that provides **income protection** to gig workers.

Instead of traditional insurance, our system:

* Uses **AI to assess risk**
* Works on a **weekly pricing model**
* Automatically **triggers payouts during disruptions**
* Requires **no manual claim process**

---

## 🎯 Target Persona

* Food Delivery Partners (Zomato / Swiggy)
* Operating in urban environments
* Dependent on daily earnings

---

## ⚙️ Key Features

### 🤖 AI-Powered Risk Assessment

* Predicts disruption probability
* Uses weather, pollution, and historical data

### 💰 Weekly Pricing Model

* Premium based on income and risk
* Simple and transparent calculation

### ⚡ Parametric Automation

* Real-time monitoring of external conditions
* Automatic payout triggers
* No manual claim required

### 🛡️ Intelligent Fraud Detection

* Location validation
* Activity tracking
* Anomaly detection

---

## 🧠 Tech Stack

* **Frontend:** React
* **Backend:** Flask / Node.js
* **Database:** MongoDB / MySQL
* **AI/ML:** Python (scikit-learn)
* **APIs:** Weather API, Pollution API
